山崎是日本第一座威士忌蒸餾廠，懷著對威士忌的滿腔熱情，對製造威士忌的文化扎根有著很大的貢獻。其高雅、豐富的口感，溫潤的東方風味，讓全世界品酒行家盛讚不已。

山崎 單一麥芽日本威士忌

豐富多彩原酒中，調和師精心挑選理想的麥芽原酒調配而成。溫和、豐富多層次的香氣、香甜滑順口感為其特點。

    Tasting Note

  • 色澤亮紅的琥珀色
  • 香氣苺果、櫻桃香
  • 口感蜂蜜的香甜風味逐漸在口腔中擴散
  • 尾韻甘甜香草香、肉桂香讓人感到愉悅的餘韻

山崎 12年 單一麥芽日本威士忌

1984年山崎蒸餾廠60周年時所誕生，是日本第一支也是最具代表性的單一麥芽威士忌，呈現日本獨有水楢桶的淡雅。

    Tasting Note

  • 色澤金黃色
  • 香氣草莓、櫻桃與日本線香的氣息
  • 口感覆盆子、白桃果香風味並帶點椰子
  • 尾韻香草甜味，清澈餘韻伴隨一絲肉桂香
